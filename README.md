# Role Sentry
Role Sentry is a web UI dashboard for monitoring role sizes and refactor status.
## Features
* Lists all roles with size metrics and a 'refactor' button
* Clicking refactor shows a preview plan before execution
* Audit logs are viewable and downloadable
## Usage
1. Run the application using `python -m pytest`
2. Access the dashboard at `http://localhost:8080` (not implemented)
